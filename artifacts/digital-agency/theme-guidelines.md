# Zystra Theme Guidelines

## Brand Colors

These are the primary brand colors to be used exclusively throughout the website:

- **Dark Purple**: `#33015a`
- **Vibrant Purple**: `#6e019c`
- **Medium Purple**: `#561d9a`
- **Black**: `#000000`
- **White**: `#ffffff`

## HSL Values (for Tailwind CSS variables)
- `#33015a` -> `hsl(274, 98%, 18%)`
- `#6e019c` -> `hsl(282, 98%, 31%)`
- `#561d9a` -> `hsl(267, 68%, 36%)`

## Usage Rules
- Do not use default Tailwind colors (e.g., indigo, pink, blue) for brand elements.
- Use the vibrant purple for primary actions, buttons, and highlights.
- Use the dark purple for deep backgrounds, dark mode surfaces, or strong contrasts.
- Use the medium purple for secondary accents and hover states.
- Use white and black for neutral text, backgrounds, and structure.
